Hiking.pt

Previous Next

This is the final project for the HarvardX MOOC CS50: Introduction to Computer Science. Hiking.pt is a website to plan hiking holidays in Portugal. It features an overview of hiking regions and maps with trails patronized by ICNF aiming to make information more accessible by showing the local distribution of trails in the regions. The website has a backoffice to edit its content.

Technologies used:
  • HTML, CSS, Javascript
  • PHP, MySQL
  • Google Maps API, Google Places API
